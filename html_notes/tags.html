Attributes is used within the opening tag of an html element. They provide additional information about html elements
1) alt Attributes- altenative text for an image if image cannot be displayed
2) width and height- is used to adjust the width and height of an image(in pixels).
3) id attributes- The id attribute in HTML assigns a unique identifier to an element, allowing it to be targeted by CSS and JavaScript for styling and manipulation purposes.
4) title attribute- The title attribute is used to explain an element by hovering the mouse over it. The behavior differs with various elements but generally, the value is displayed while loading or hovering the mouse pointer over it.
5) href attribute- The href attribute in HTML, used with the tag, specifies a link destination. Clicking the linked text navigates to this address. Adding `target=”_blank”` opens it in a new tab.
6) style attributes- The style attribute is used to provide various CSS effects to the HTML elements such as increasing font-size, changing font-family, coloring, etc.
